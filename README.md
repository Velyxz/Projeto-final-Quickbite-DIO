# Projeto-final-Quickbite-DIO
QuickBite – UX/UI Prototype  QuickBite é um protótipo de aplicativo mobile de delivery desenvolvido como parte de um desafio de UX/UI. O projeto inclui briefing, Design System, wireframes de alta fidelidade e protótipo navegável no Figma, com foco em uma experiência simples, intuitiva e eficiente para realizar pedidos online.
