#  QuickBite – UX/UI Delivery App Prototype

QuickBite é um protótipo de aplicativo mobile de delivery desenvolvido como parte de um desafio de UX/UI Design. O projeto simula uma plataforma de pedidos online, permitindo que usuários encontrem restaurantes, explorem cardápios, adicionem itens ao carrinho e acompanhem o status do pedido de forma simples e intuitiva.

O objetivo do projeto foi aplicar, de forma prática, os principais fundamentos de **UX Research, Design System, Wireframing e Prototipagem**, simulando o desenvolvimento de um produto digital real.

 Objetivo do Projeto

Criar um protótipo funcional de um aplicativo de delivery que priorize:

* Experiência do usuário simples e intuitiva
* Redução de fricção no fluxo de pedidos
* Hierarquia visual clara
* Navegação eficiente
* Consistência visual através de um Design System

 Processo de Design

O projeto foi desenvolvido seguindo etapas comuns do processo de **UX/UI Design**:

 1. Briefing do Projeto

Definição do problema, público-alvo e objetivos da aplicação.

 2. Design System

Criação de um sistema visual consistente contendo:

* Paleta de cores
* Tipografia
* Espaçamentos
* Componentes reutilizáveis
* Botões, inputs e cards

 3. Wireframe de Alta Fidelidade

Estruturação das telas principais do aplicativo com foco em usabilidade e organização da informação.

 4. Prototipagem

Criação de um protótipo navegável simulando a jornada completa do usuário dentro do aplicativo.

 5. Estrutura de Fluxo

Definição do fluxo principal de navegação do usuário:

Splash Screen
→ Login
→ Home
→ Restaurante
→ Carrinho
→ Checkout
→ Rastreamento do Pedido

📱 Principais Telas do Protótipo

* Splash Screen
* Login / Cadastro
* Home com lista de restaurantes
* Página de restaurante e cardápio
* Carrinho de compras
* Checkout
* Rastreamento de pedido

 Design System

O projeto inclui um Design System estruturado para garantir consistência visual e escalabilidade.

Principais elementos:

* Sistema de cores
* Hierarquia tipográfica
* Componentes reutilizáveis
* Grid baseado em 8pt
* Espaçamento padronizado


 Ferramentas Utilizadas

* Figma
* Prototipagem interativa
* Componentes e Auto Layout
* Variants e Design System


 Sobre o Desafio

Este projeto foi desenvolvido como parte de um desafio prático de UX/UI Design, com o objetivo de aplicar conceitos de design de produto digital na criação de um protótipo funcional.
